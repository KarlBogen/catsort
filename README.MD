#	Komfortable Kategoriesortierung

*Getestet mit Shopversionen 3.1.6*

## Beschreibung

Mit diesem Modul können Kategorien per "Drag and Drop" sortiert werden.
<br>
Es kann nur innerhalb der Kategorieebene verschoben werden!
<br>
Eingeklappte Kategorien erhalten ebenfalls eine Sortierung!
<br>
<br>
Sobald das Systemmodul installiert ist steht Sortierung unter dem Menüpunkt **Katalog -> Komfortable Kategoriesortierung** zur Verfügung.
<br>

## Installation

1. BACKUP MACHEN!!!

2. Dateien aus dem Ordner **new_files/** in den Shop kopieren (es wird nichts überschrieben)
   <br>Hinweis: Der Name des Ordners "admin" muss an den des Shops angepasst werden!

3. Im Adminbereich unter "Module->System Module" findet man jetzt den Eintrag "Komfortable Kategoriesortierung".
   Dieses Modul auswählen und installieren.

   Das Modul ist nun installiert und aktiviert.


## Deinstallation

1. Im Adminbereich unter "Module->System Module" den Eintrag "Komfortable Kategoriesortierung" auswählen.

2. Modul deinstallieren
   Hinweis: Damit werden alle Datenbankeinträge, keine Dateien, gelöscht.

3. Wer auch die Dateien entfernen möchte kann den roten Button "Moduldateien löschen" klicken.
